## :computer: Deployment
Deploy the app using XAMPP, put all files in   
```
/xampp/htdocs/(webname)
```

- Start Apache and PHPMyAdmin   
- Create Database `aptel_db` in PHPMyAdmin
- Import `.sql` file in `/sql/` 
- Go to http://localhost/(webname)/
